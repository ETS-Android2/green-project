get_readings Request
This request has the information about the previous readings of the last hours, some of the data contains the date, fruit, the weight's value, and also it registers the color of every fruit in RGB format.
get_fruits
This request has all the relative information about the fruits, that contains the own fruit's code to identify the fruit, a very short description and the name of the fruit, all the data in this request is Obtained from the view VW_readings_last_hour
